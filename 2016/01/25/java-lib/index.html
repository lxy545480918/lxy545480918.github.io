JAVA常用类库

发表于 2016-01-25 | 分类于 收藏
  1. Joda：时间处理类库
  2. Jackson：json处理
  3. Guava：集合、缓存、支持基本类型、并发函数库、通用注解、字符串处理、I/O等。
  4. Apache Commons：提供各种用途的函数，比如配置、验证、集合、文件上传或XML处理等。
  5. crawler4j：java爬虫框架。
